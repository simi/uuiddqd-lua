# Uuiddqd-lua

Simple libuuid lua c binding

## Installation

```
make
```

## Usage

```lua
require("uuiddqd")
print(uuid_generate()) #=> "c3523d95-ecb5-4f86-9f75-aa7ac230e8f0"
```
